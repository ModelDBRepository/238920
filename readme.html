The simulation and plotting code for the paper:

Hagen E, Fossum JC, Pettersen KH, Alonso JM, Swadlow HA, Einevoll GT (2017) Focal Local Field Potential Signature of the Single-Axon Monosynaptic Thalamocortical Connection. J Neurosci 37:5123-5143
http://dx.doi.org/10.1523/JNEUROSCI.2715-16.2017

is available at the authors github repository:

https://github.com/LFPy/monosynaptic_LFP

There are other tools that the paper used that are available in github (see paper for github links).
